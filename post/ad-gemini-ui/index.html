预约 · Gemini UI
发布于 2021-08-05，包含标签 # Gemini UI# PPT OS# 氧生态。

预告 · 主要介绍

Gemini UI 是由 Lipus Studio 制作的、基于 Microsoft PowerPoint 的仿操作系统。系统 UI 使用以 Simple、Smooth 为设计核心的 G-Level Design 设计语言，界面好看、简洁，动画以平滑和流畅为主。
“不搞花里胡哨的东西、削减华而不实的功能、让系统中的各个应用程序都有其用。”，是系统的一大特点，也是我们在 PowerPoint 的诸多限制中，对系统制作的要求。

目前系统的 Beta Preview 仍在制作中，10 月初旬即刻放送，明年将放送正式版，敬请期待！
